# Frontend

Vite + React + TypeScript UI for **Higher School of Poker**.

## Run (dev)

1) Start backend on `http://localhost:8000`
2) In `frontend/`:

```bash
npm i
npm run dev
```

## Environment

- `VITE_API_URL` (default: `http://localhost:8000`)
- `VITE_WS_URL` (optional; by default derived from `VITE_API_URL`)
